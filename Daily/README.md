# Daily Logs

This folder stores one markdown log per practice day or problem session.

Recommended rhythm:

1. Create a new problem file with the helper script.
2. Solve the problem with a `No AI First Attempt`.
3. Fill in the matching daily log before ending the session.
4. Revisit mistakes and rebuild key solutions from memory.

Suggested naming:

- Python solution: `topic/problem_name.py`
- Daily log: `Daily/YYYY-MM-DD-problem-name.md`
